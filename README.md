# 🕷️ Homem-Aranha: Através do Aranhaverso (Spider-Verse)

> Um projeto web interativo e animado, desenvolvido com HTML5, CSS3 e JavaScript puro (Vanilla JS). O objetivo principal deste projeto foi aprender e praticar técnicas de animação web, manipulação do DOM e a integração de componentes visuais imersivos, como galerias de mídia.

## 🎯 Objetivos do Projeto

Este repositório foi criado com foco em fins educacionais e evolução técnica pessoal. Os principais pontos desenvolvidos foram:
1. **Domínio de Animações CSS:** Criação de efeitos visuais dinâmicos que dão vida à interface, inspirados na estética vibrante dos filmes.
2. **Fluidez e Responsividade:** Garantir que a experiência de navegação e as transições permaneçam consistentes em diferentes resoluções de tela.
3. **Integração de Bibliotecas de Terceiros:** Implementar e customizar o Fancybox para enriquecer a experiência de mídia do usuário.
